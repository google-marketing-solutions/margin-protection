# TypeScript Developer Documentation

This section provides in-depth guides and examples for developers working on the Margin Protection TypeScript codebase.

## Custom Rule Examples

For a practical guide on how to implement new monitoring rules, please see the platform-specific examples below. Each document provides a complete, step-by-step walkthrough of creating a real-world rule.

- **[DV360 Rule Examples](./examples_dv360.md):** Learn how to create a rule that checks if a DV360 insertion order is pacing ahead or behind its budget based on the flight dates.
- **[SA360 Rule Examples](./examples_sa360.md):** A guide to building a rule that alerts on low Quality Score keywords in a Search Ads 360 account.
- **[Google Ads Rule Examples](./examples_googleads.md):** An example of how to monitor the Cost Per Acquisition (CPA) for specific audiences in a Google Ads campaign.
- **[CM360 Rule Examples](./examples_cm360.md):** An example of how to monitor for active placements that are missing ad assignments.
