Class: AppsScriptFrontend

AppsScriptFrontend(category, injectedArgs)

The front-end for Apps Script UIs. This is extensible for customer use-cases. The easiest approach to modifying this is to handle changes in injectedArgs. It is also possible to extend this class for more in-depth changes.
